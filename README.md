# PoolTech CRM

CRM system for pool maintenance services with integrated pricing calculator.

## Features

- **Dashboard**: Overview of clients, scheduling, and financial metrics
- **Client Management**: Track clients, contact information, and service history
- **Appointment Scheduling**: Manage and schedule service appointments
- **Quote Management**: Generate and track service quotes
- **Financial Tracking**: Monitor revenue, expenses, and profitability
- **Preço Justo Calculator**: Intelligent pricing calculator with cost-based pricing
  - Dynamic price calculation based on costs and hours worked
  - Minimum price protection (cost × 1.05)
  - Ideal price suggestion (cost + hours × rate × 1.2)
  - Historical tracking of calculations

## Getting Started

1. Open `index.html` in a web browser
2. All data is stored locally in your browser (localStorage)
3. No backend server required

## Technology Stack

- HTML5
- CSS3 (Dark theme with modern UI)
- Vanilla JavaScript (ES6+)
- Chart.js for data visualization
- Intl API for currency formatting (BRL)
